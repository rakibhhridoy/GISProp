\section{Regulatory Framework}

CAAB Requirements Summary:
\begin{itemize}
    \item Drone Registration: >248g threshold
    \item Pilot Certification: 50+ flight hours via approved academy
    \item Flight Approval: 45-day FSR with multi-agency clearance
    \item Insurance: BDT 50-100L liability minimum
    \item Operating Hours: Daylight only, Visual Line of Sight required
\end{itemize}
