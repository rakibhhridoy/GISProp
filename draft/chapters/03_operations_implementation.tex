\chapter{Operations, Regulatory \& Implementation}

\section{CAAB Regulatory Requirements}

Bangladesh's drone regulations are clear and achievable through the Civil Aviation Authority Bangladesh (CAAB):

\subsection{1. Drone Registration}

\begin{itemize}
    \item \textbf{Required for:} Drones >248g or range >100m
    \item \textbf{Process:} Submit specs, insurance proof, operator credentials
    \item \textbf{Timeline:} 2-3 weeks
    \item \textbf{Cost:} BDT 10-20K per aircraft
\end{itemize}

\subsection{2. Remote Pilot Certification (ROC)}

\begin{itemize}
    \item \textbf{Training Partner:} Avion Aerospace Academy (CAAB-approved)
    \item \textbf{Requirements:} 50+ flight hours, ground school, Class 3 medical cert, written + practical exam
    \item \textbf{Duration:} 4-6 weeks per pilot
    \item \textbf{Cost:} BDT 8-12L per pilot (includes all training)
    \item \textbf{Benefit:} 160+ pilots already trained by Avion Academy
\end{itemize}

\subsection{3. Flight Safety Request (FSR) \& Security Clearances}

\begin{itemize}
    \item \textbf{Submission:} 45 days in advance per mission
    \item \textbf{Approvals:} ADOC (airspace), DGFI (security), NSI (intelligence), Local Police
    \item \textbf{Timeline:} Typically 30-45 days
    \item \textbf{Strategy:} Plan missions in batches to optimize approval process
\end{itemize}

\subsection{4. Insurance \& Operational Restrictions}

\begin{itemize}
    \item \textbf{Third-Party Liability Insurance:} Mandatory, BDT 50-100L coverage minimum
    \item \textbf{Premium:} BDT 3-5L annually
    \item \textbf{Altitude:} Max 120m AGL for "Specific" operations
    \item \textbf{Line of Sight:} Visual contact required at all times
    \item \textbf{Hours:} Daylight operations only (30 min before sunrise to 30 min after sunset)
    \item \textbf{Wind:} No operations in wind >10 m/s
\end{itemize}
